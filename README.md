## Install script on Termux
```
$ pkg update && pkg upgrade
$ pkg install python2
$ pip2 install requests
$ pip2 install mechanize
$ pkg install git
$ git clone https://github.com/anggaxd/smbf
```

## Run script
```
$ cd smbf
$ python2 crack.py
```
